### Summary Lines (`****`):
```

```

**DOM Pattern for `****` labor summary**:
```html
  ****    0200  0200                       <span style='text-decoration:underline'>   2.000</span> <span style='text-decoration:underline'>   0.000</span> <span style='text-decoration:underline'>  -2.000</span> <span style='text-decoration:underline'>    40.00</span> <span style='text-decoration:underline'>     0.00</span>   <span style='text-decoration:underline'>   -40.00</span>  FABRICATION LAYOUT
```

**6 `<span>` tags per `****` line, always `style='text-decoration:underline'`**:

| Span # | Field | Example | Notes |
|--------|-------|---------|-------|
| 1 | EST HRS | `   2.000` | **This is the estimated hours field** |
| 2 | ACTUAL HRS | `   0.000` | Actual hours worked |
| 3 | VARIANCE (hrs) | `  -2.000` | actual - estimated |
| 4 | EST COST | `    40.00` | Estimated cost ($) |
| 5 | JOB COST | `     0.00` | Actual job cost ($) |
| 6 | COST VARIANCE | `   -40.00` | job_cost - est_cost |

**Critical Note**: When EST HRS is blank (no estimate set), the span contains only whitespace:
```html

```
The parser correctly handles this as `None`.

## 6. Universal `<span>` Pattern

**Every** `****` summary line across all three sections (Labor, Material, Outplant) uses the identical DOM pattern:

```html
<span style='text-decoration:underline'>{VALUE}</span>
```

- Always exactly `style='text-decoration:underline'` (single quotes, no class)
- Always 6 spans per `****` line
- Blank values = whitespace-only content inside span
- Values are right-justified within the span text, space-padded
- Negative values have leading `-`
- Thousands use commas (e.g. `3,215.00`)

### Universal Extraction Pattern:
```python
from bs4 import BeautifulSoup, Tag

def extract_star_line(line_text_with_html):
    """Extract 6 numeric values from a **** summary line's <span> tags."""
    soup = BeautifulSoup(line_text_with_html, 'html.parser')
    spans = soup.find_all('span', style='text-decoration:underline')
    values = []
    for span in spans:
        text = span.get_text().strip().replace(',', '')
        if text:
            values.append(float(text))
        else:
            values.append(None)
    return values  # [est, actual, variance, est_cost, job_cost, cost_variance]
```
