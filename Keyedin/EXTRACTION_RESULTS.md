## Access Paths Discovered

### 1. Browse Spooled Reports (REPORT.VIEW.INDEX)
- **67 reports** found in the spool queue
- Reports are generated by BI functions and stored server-side
- Can be viewed anytime through REPORT.VIEW
- Content is in `<pre>` formatted text blocks within iframes
- **Extraction method**: Playwright CDP → iframe content → parse pre blocks

### 2. ERP Browser (272 accessible functions)
- BradyF has access to **19 modules** and **272+ functions**
- Modules: CRM, Project Management, Estimating, Sales, Shipping, Purchasing, Inventory, Job Cost, Production, A/R, A/P, MRP, and more
- BI reports generate spooled output accessible via REPORT.VIEW
- Listings (ITEM.MASTER.LIST, VM.INQUIRY, etc.) render directly in HTML tables

## Key Technical Findings

1. **Session management**: Cookie-based (SESSIONID, ASP.NET_SessionId, user, secure)
2. **MVI_KEY**: Dynamically generated per APPLOAD request, required for form submissions
3. **Report generation**: Asynchronous — VIEW.LOG tracks progress, REPORT.VIEW stores output
4. **License quota**: 10 concurrent users, occasionally blocks login
