How to use the dialog window "Circ/Sq/Rect Foundations"

This dialog window is shown in Figure 1. It allows to calculate circular/square/rectangular foundation parameters.

For calculating square foundation parameters select radio button "Square" in group "Caisson type".

If any term that is used in this dialog box is unknown to you, please refer to glossary of terms.

For getting results, enter your data to edit boxes, select steel type that you intend to use, change soil pressure values to desired ones and click on the button Calculate. After a few seconds (depends on your machine speed) you shall receive results in the rightmost window under title "Results".

For selecting the proper soil type click on the button "Change Selected Soil Pressure". New dialog window "Change soil pressure" shall appear. If you need some help in determining your soil type, refer to "Understanding soil pressure" article.

Note that you must enter base width into edit box with label "Base Diameter /Width".

If your requirements are too severe, application may fail to find a solution. You will be warned about this. In this case you shall need a special engineering.

For proper interpreting of recommended pipe sizes refer to "Calculating perimeter bending force" article.

Results and information about each sign on list may be printed. For this, click on the button Print Summary. Printer setup dialog window shall appear, where you can select appropriate printer. Then click OK, and here you are.

For your convenience all entered information and calculated results may be removed from dialog window. For this, click on the button Clear, and controls in dialog window shall be reset to appropriate default values.
